# Autobidder Frontend

React + TypeScript frontend for managing the Autobidder application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Make sure the backend API server is running on port 8000:
```bash
python api_server.py
```

The frontend will be available at http://localhost:3000

## Features

- **Dashboard**: View statistics and control the autobidder
- **Bids**: View all bid history
- **Config**: Edit configuration settings
- **Prompt**: Edit the prompt template used for generating bid messages
- **Skills**: Manage your skills list

## Build

To build for production:
```bash
npm run build
```
